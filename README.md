# CheapSat
### DIY一颗人造卫星所需要的购物清单
本清单的所有商品均可在中国大陆地区获得，主要通过淘宝购买。
本清单仅供科普教学，真正合理的卫星系统可能需要更多的投入，无论是脑力层面的还是经济层面的。
